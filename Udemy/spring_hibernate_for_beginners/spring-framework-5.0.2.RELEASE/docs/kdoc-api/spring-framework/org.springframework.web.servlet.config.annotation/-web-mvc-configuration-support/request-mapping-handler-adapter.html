requestMappingHandlerAdapter

@Bean open fun requestMappingHandlerAdapter(): RequestMappingHandlerAdapter

Returns a RequestMappingHandlerAdapter for processing requests through annotated controller methods. Consider overriding one of these other more fine-grained methods:

  • #addArgumentResolvers for adding custom argument resolvers.
  • #addReturnValueHandlers for adding custom return value handlers.
  • #configureMessageConverters for adding custom message converters.
